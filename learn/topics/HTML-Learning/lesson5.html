JavaScript Basics

Turn Your Website Into an Interactive Experience

🧠 What JS Does

JavaScript is the “brain” of your website. It responds to user actions and updates the page dynamically.

  • Responds to clicks
  • Validates forms
  • Changes HTML/CSS dynamically
  • Creates interactive UI elements

⚡ Basics: Variables

// Store data let name = "Alice"; const pi = 3.14; var age = 25;

Use let or const most of the time.

🔹 Functions

function greet(user) { alert("Hello " + user); } greet("Alice");

Functions allow reusability and cleaner code.

🔗 Events (Real Interaction)

JS reacts to actions like clicks:

User clicks → JS runs → DOM updates. That’s the power.

📄 Manipulating HTML

Old Text

JS can change text, colors, visibility, or add/remove elements.

🧪 Exercise (25–30 mins)

Add interactivity to your previous page:

  • Click a button → change heading text
  • Change background color dynamically
  • Log messages in console

🔥 Challenge (40–50 mins)

Create a simple interactive profile card:

  • Click “Show Bio” → reveals hidden paragraph
  • Click “Change Color” → updates card background
  • Use functions + event listeners

🧠 Key Takeaways

  • JS = interactivity + dynamic content
  • Variables + functions = building blocks
  • Events connect user → website
