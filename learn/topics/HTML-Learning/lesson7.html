Final Project

Build Your First Mini Portfolio Website

🎯 Project Goal

Using HTML, CSS, JS, and Flexbox, build a portfolio website with the following sections:

  • Header with logo + navigation
  • About section
  • Projects section (Flexbox cards)
  • Contact form (interactive)
  • Footer with social links

🧱 Header & Navigation

<header class="flex-demo"> <div class="logo">My Portfolio</div> <nav> <a href="#about">About</a> <a href="#projects">Projects</a> <a href="#contact">Contact</a> </nav> </header>

Flexbox makes logo + menu align perfectly on one line.

📦 Projects Section

<section id="projects" class="flex-demo"> <div class="project">Project 1</div> <div class="project">Project 2</div> <div class="project">Project 3</div> </section>

Use Flexbox to align project cards with spacing and wrapping.

📝 Contact Form (Interactive)

<form id="contactForm"> <input type="text" placeholder="Name" id="name"> <input type="email" placeholder="Email" id="email"> <textarea placeholder="Message" id="message"></textarea> <button type="submit">Send</button> </form> <script> document.getElementById("contactForm").addEventListener("submit", function(e){ e.preventDefault(); alert("Thank you, " + document.getElementById("name").value + "!"); }); </script>

The form is fully interactive using JS & DOM.

🧪 Exercise (1 Hour)

Build your site step by step:

  • Create header + navigation
  • Add an About section with text + image
  • Make a Projects section using Flexbox
  • Add a Contact form and handle submission with JS
  • Add a footer

🔥 Challenge (1–2 Hours)

Enhance your portfolio:

  • Add hover effects to project cards
  • Use JS to dynamically add projects to the section
  • Validate the form input (required fields)
  • Make it responsive on mobile screens

🧠 Key Takeaways

  • You can now build a complete website from scratch
  • HTML + CSS + JS + Flexbox = complete toolkit
  • Project work = true learning and retention
