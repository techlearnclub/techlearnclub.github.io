CSS Fundamentals

Make Your Websites Actually Look Good

🧠 What CSS REALLY Does

HTML gives structure. CSS controls appearance. But more importantly — CSS controls perception.

  • Spacing = readability
  • Color = emotion
  • Layout = usability

Bad CSS = ugly and unusable websites.

🎯 How CSS Works

selector { property: value; }

Example:

h1 { color: blue; }

This selects ALL h1 elements and styles them.

🎯 Selectors (IMPORTANT)

h1 { } .className { } #idName { }
  • Element = targets all elements
  • Class = reusable
  • ID = unique

Professionals mostly use classes.

📦 The Box Model (CRITICAL)

Every element is a box:

  • Content
  • Padding (inside space)
  • Border
  • Margin (outside space)
This box has padding and margin

If you don’t understand this, layouts will break.

🎨 Colors & Backgrounds

body { background: #0b0f19; color: white; }

Use contrast. Dark background + light text = readable.

🔠 Typography

body { font-family: Arial; line-height: 1.6; }

Good typography = professional look.

📏 Spacing (UNDERRATED SKILL)

div { margin: 20px; padding: 20px; }

Spacing is what separates beginner vs pro design.

🧪 Exercise (25 mins)

Take your previous HTML page and:

  • Add background color
  • Add spacing (margin/padding)
  • Change text colors

Make it clean, not random.

🔥 Challenge (40 mins)

Design a profile card:

  • Image

Focus on spacing + colors + layout.

🧠 Key Takeaways

  • CSS = visual system, not decoration
  • Box model is everything
  • Spacing > fancy effects
