Layouts & Flexbox

Build Modern Website Layouts

🧠 Why Layouts Matter

Without proper layouts, websites look broken. Flexbox allows professional alignment without hacks.

  • Proper spacing
  • Centering elements
  • Responsive design thinking

📦 Flexbox Basics

Use display: flex to create flexible containers.

.container { display: flex; }

Children align automatically in a row by default.

Box 2
Box 3

⚙️ Flex Properties

  • justify-content → horizontal alignment
  • align-items → vertical alignment
  • flex-direction → row / column
  • gap → spacing between elements
.container { display: flex; justify-content: center; align-items: center; gap: 20px; }

💡 Real-World Example

Header with logo + menu:

<div class="logo">Logo</div> <nav>Menu</nav>

Flex aligns them perfectly without floats.

🧪 Exercise (25–30 mins)

Take a section of your page and:

  • Make it a Flex container
  • Align items horizontally
  • Use gap to space elements
  • Try vertical alignment with align-items

🔥 Challenge (40 mins)

Build a small "feature section" with 3 boxes:

  • Each box has title + description
  • Use Flexbox to align them evenly
  • Make it responsive: stack on small screens

🧠 Key Takeaways

  • Flexbox solves layout issues elegantly
  • Align and space without hacks
  • Foundation for responsive modern sites
